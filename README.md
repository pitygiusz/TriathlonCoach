# Triathlon Coach AI

A personal triathlon training assistant built with **Streamlit** and **Google Gemini**. The app helps athletes log workouts, track progress, and receive  AI-generated weekly training plans.

## Goal

The goal of this project is to support triathlon preparation by elevating simple in-chat advice from AI into a more personalized and actionable coaching experience. By combining structured workout tracking with AI-driven insights, the app provides tailored weekly plans that consider your recent training history, current fatigue, upcoming race goals, and local weather to generate a personalized weekly plan.

## Key Features

- **Workout logging** - Record training sessions with discipline, duration, distance, RPE, and notes. Manage and delete entries through an interactive table.
- **Training statistics** - View total time and distance, per-discipline breakdowns, and a timeline chart of all workouts.
- **AI coaching** - Get a weekly training review and a suggested 7-day plan generated by Google Gemini, with optional context: race goals, weather forecast, long-term stats, and reduced-load weeks. 
- **Prompt engineering** - The coaching prompts are carefully engineered to produce structured, consistent output by combining dynamic athlete data with fixed persona instructions, preferences, and formatting requirements to guide the model toward coach-like responses.
- **Additional tools** - Beyond the core features, the app includes:
    - Nutrition assistant
    - Gym plan generator 
- **Race countdown** - Dashboard counters showing days remaining to target races.
- **Weather integration** - Fetches a 7-day weather forecast via Open-Meteo API to inform outdoor training recommendations.
- **Self-hosted on Raspberry Pi** - The app is designed to run continuously on a Raspberry Pi. Combined with Tailscale, it can be accessed securely from any device, anywhere, without exposing it to the public internet.


## Project Structure

```
TriathlonAI/
├── app.py          # Streamlit UI - all tabs, forms, and user interaction
├── coach.py        # AI logic - prompt construction and Gemini API calls
├── database.py     # SQLite database layer - workouts and coach log persistence
├── tools.py        # External tools - weather forecast and long-term stats aggregation
└── triathlon_logs.db  # Auto-created SQLite database (not committed)
```

## How to Run

**1. Install dependencies**

```bash
pip install streamlit google-genai pandas requests
```

**2. Set up your Gemini API key**

The app uses the Google Gemini API. Make sure your API key is available as an environment variable:

```bash
export GOOGLE_API_KEY="your_api_key_here"
```

**3. Start the app**

```bash
streamlit run app.py
```

The app will open in your browser at `http://localhost:8501`.
